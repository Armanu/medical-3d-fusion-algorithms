This is a demo code for the multi-focus image fusion algorithm described in the following paper:

M. Nejati, S. Samavi, S. Shirani, "Multi-focus Image Fusion Using Dictionary-Based Sparse Representation", Information Fusion, vol. 25, Sept. 2015, pp. 72-84. 

The code and the algorithm are for academic use only.
Please refer to the publication above if you use this software.


==================
 Notice 					 	 
==================
This code has been tested under Matlab R2012b in Windows 7 64-bit.


==================
 Dependencies
==================
1. OMPBox10 which is available at: 
   http://www.cs.technion.ac.il/~ronrubin/software.html

2. Guided filtering code from Dr. Kaiming He's webpage 
   (http://kaiminghe.com/eccv10/guided-filter-code-v1.rar).
	
3. Graph-cut optimization software (gco-v3.0) from
   http://vision.csd.uwo.ca/code
